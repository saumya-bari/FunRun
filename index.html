FUN RUN
HOW TO PLAY
Controls:
• Left/Right Arrows or Swipe: Switch Lanes
• Up Arrow/Space or Swipe Up: Jump over hurdles
• Down Arrow or Swipe Down: Slide under debris

Game Modes:
Tap FUN MODE in the top bar to switch between the classic Astronaut runner and the neon Rocket mode!
DEVELOPER INFO
FUN RUN is an endless runner set in deep space. Dodge meteors, collect stardust, and use cosmic boosters to achieve the highest score!

Game Controls:
• Left/Right Arrows: Switch Lanes
• Up Arrow/Space: Jump
• Down Arrow: Slide
• Touch/Swipe: Swipe directions to move
